# Dog Bark Detection

## Conda Environment

```bash
conda env create -f environment.yml
conda activate dog-bark
```

### Run Prediction

```bash
python3 main.py
```

## Docker

```bash
docker build -t dog-bark .
docker run dog-bark
```
